Why Oxide

Everything you need.
Nothing you don't.

A Rust-native web framework that compiles to WebAssembly with fine-grained reactivity, zero JavaScript overhead, and full type safety.

🎯

Fine-Grained Reactivity

Signals track exactly what changes. No virtual DOM diffing. O(1) updates directly to the DOM nodes that need them.

🚀

Zero JavaScript

Write your entire frontend in Rust. Type-safe, memory-safe, compile-checked. The only JS is the WASM loader shim.

📦

Tiny Bundles

30 KB hello world. 178 KB for a full 18-demo showcase. Tree-shaking at the Rust compiler level.

⚡

Modern DX

JSX-like view! macro with if/else, for loops, bind:value, class toggling. Feels like React, runs like C.

🛡️

Type Safe

Catch bugs at compile time, not in production. Strongly-typed signals, props, and events. No runtime surprises.

🔧

Full Web API Access

Canvas, Fetch, LocalStorage, Drag & Drop, Clipboard, SVG, Audio — every browser API, from Rust.

Counter — Source
let mut count = signal(0);

view! {
    <div>
        <p class="big-num">{count}</p>
        <div class="row">
            <button on:click={move |_| count -= 1}>"−"</button>
            <button on:click={move |_| count.set(0)}>"Reset"</button>
            <button on:click={move |_| count += 1}>"+"</button>
        </div>
    </div>
}
