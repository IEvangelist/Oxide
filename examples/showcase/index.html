Why Oxide

Everything you need.
Nothing you don't.

A Rust-native web framework that compiles to WebAssembly with fine-grained reactivity, zero JavaScript overhead, and full type safety.

🎯

Fine-Grained Reactivity

Signals track exactly what changes. No virtual DOM diffing. O(1) updates directly to the DOM nodes that need them.

🚀

Zero JavaScript

Write your entire frontend in Rust. Type-safe, memory-safe, compile-checked. The only JS is the WASM loader shim.

📦

Tiny Bundles

30 KB hello world. 178 KB for a full 16-demo showcase. Tree-shaking at the Rust compiler level.

⚡

Modern DX

JSX-like view! macro with if/else, for loops, bind:value, class toggling. Feels like React, runs like C.

🛡️

Type Safe

Catch bugs at compile time, not in production. Strongly-typed signals, props, and events. No runtime surprises.

🔧

Full Web API Access

Canvas, Fetch, LocalStorage, Drag & Drop, Clipboard, SVG, Audio — every browser API, from Rust.

Get Started

From zero to WASM
in five steps.

Everything you need to build and ship your first Oxide app.

1

Install Rust & WASM Toolchain

$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
$ rustup target add wasm32-unknown-unknown
$ cargo install wasm-pack
2

Create Your Project

$ cargo new --lib my-app
$ cd my-app
3

Add Dependencies

Cargo.toml 
[lib]
crate-type = ["cdylib"]

[dependencies]
oxide = { git = "https://github.com/IEvangelist/Oxide" }
wasm-bindgen = "0.2"
4

Write Your App

src/lib.rs 
use oxide::prelude::*;
use wasm_bindgen::prelude::*;

#[wasm_bindgen(start)]
pub fn main() {
    mount("#app", || {
        let mut count = signal(0);
        view! {
            <button on:click={move |_| count += 1}>
                "Clicks: " {count}
            </button>
        }
    });
}
5

Build & Run

$ wasm-pack build --target web
$ python -m http.server 8080
